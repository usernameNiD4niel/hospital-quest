## Local Storage
-- `currentIndex`: This will be use to index the user avatar.
-- `name`: User name.
-- `progress`: This is the overall progress of the user from the departments and QA.
```json
{
    "progress": [
        {
            "department": "Operating Room",
            "stars": 2
        },
        // ...
    ],
    "currentDepartment": "Operating Room",
    "totalStars": 2
}
```
